第二课

textbook: https://ttrw.jp/static/textbook//grammer_free/19/dr1-2.pdf

文型 (ぶんけい - bunkei)

これは　辞書です

これは　コンピューターの　本です

それは　私の　傘です

この　傘は　私のです

  • これは／それは／あれは

    • これ = 这（在说话人身边）
    • それ = 那（在听话人身边）
    • あれ = 那（离双方都远）

  • の（名词的修饰关系）

    • 「の」 (Content)
      • これ は コンピューターの 本 です。
      • This is a book on computers (a computer book).
      • It tells us what kind of object Noun 2 is, or what its contents are.
    • 「の」 (Possession)

      • それ は 私の 傘 です。

      • That is my umbrella.

      • This is the most common use of the particle 「の」. It indicates ownership. Noun 1 owns Noun 2.

例文 (れいぶん - reibun)

これはテレホンカードですか。

･･･はい、そうです。


それは　ノートですか。

･･･いいえ、そうじゃありません。手帳です。


それは 何ですか。

･･･[これは] 名刺です。


これは「9」ですか、「7」ですか。

･･･「9」です。


それは何の雑誌ですか。

･･･自動車の雑誌です。


あれは　だれの　かばんですか。

･･･佐藤さんのかばんです。


この傘はあなたのですか。

･･･いいえ、わたしのじゃありません。


このかぎはだれのですか。

･･･わたしのです。

2. 句子公式总结（按用途记）

  • 判断 + 提问：
    • これは N ですか。→ 这是 N 吗？
    • それは N ですか。→ 那是 N 吗？
  • 选择疑问句：
    • A ですか、B ですか。→ 是 A 还是 B？
    • これは「9」ですか、「7」ですか。
  • 询问事物是什么：
    • それは 何ですか。→ 那是什么？
    • それは 何の N ですか。→ 那是「什么的」N / 关于什么的 N？
  • 询问所属/归属：
    • あれは だれの N ですか。→ 那是谁的 N？
    • この N は あなたのですか。→ 这 N 是你的吗？
    • この N は だれのですか。→ 这 N 是谁的？
  • 肯定/否定回答模板：
    • はい、そうです。→ 是的，是这样。
    • いいえ、そうじゃありません。N です。→ 不，不是，是 N。
    • いいえ、わたしのじゃありません。→ 不，不是我的。
    • わたしのです。→ 是我的。

3. 语法点剖析

  • 指示词 + は：これは／それは／あれは

    • これ：这（在说话人身边）。
    • それ：那（在听话人身边）。
    • あれ：那（离双方都远）。
    • 后面接「は」做主题：これはテレホンカードですか。

  • 助词「の」的两大用法

    • 所有/所属：
      • だれの かばん（谁的包）、わたしの かばん（我的包）、佐藤さんの かばん（佐藤先生的包）。
      • 结构：N1 の N2 → “N1 的 N2”。
    • 内容/类别：
      • 自動車の 雑誌：汽车杂志（内容是“汽车”）。
      • コンピューターの 本：计算机的书（内容是“计算机”）。

  • 疑问词「何」「だれ」+ の

    • 何の N：什么的 N / 关于什么的 N
      • それは 何の 雑誌ですか。→ 那是关于什么的杂志？
    • だれの N：谁的 N
      • あれは だれの かばんですか。→ 那是谁的包？

  • 「の」的代词化（代替名词）

    • この傘は あなたのです。
      • 表面意思：这把伞是你的。
      • 实际结构：あなたの（傘）です，「の」代替前面的「傘」，避免重复名词。
    • このかぎは だれのですか。→ わたしのです。
      • 问句里的「だれの（かぎ）」和回答里的「わたしの（かぎ）」都省略了「かぎ」，靠「の」来代替。

  • 否定表达「〜じゃありません」

    • 相当于更正式的「〜ではありません」，是否定「〜です」的礼貌说法。
    • 例：いいえ、そうじゃありません。手帳です。→ 不，不是（笔记本），是记事本。

2. 句子功能 & 结构拆解

  • 开场确认对方身份

    • はい。どなたですか。
      • はい：喂 / 是（接电话、应答时的开场）。
      • どなた：礼貌版“谁”。
      • 〜ですか：礼貌疑问结尾。
      • 合起来：喂，请问是哪位？—— 标准“接电话/开门问身份”的表达。

  • 报上房间号 + 姓名

    • 408の サントスです。
      • N1 の N2：N1 的 N2（所属关系）。
      • 408の サントス：408 房间的 Santos（住在 408 的那位 Santos）。
      • 〜です：礼貌判断句。
      • 场景：在公寓/宿舍里表明“我是 X 房间的某某”。

  • 正式见面 + 建立关系

    • こんにちは。サントスです。
      • こんにちは：白天打招呼“你好”。
      • サントスです：我是 Santos（名词＋です）。
    • これから お世話に なります。
      • これから：今后、从现在起。
      • お世話：关照、照顾。
      • 〜に なります：变成某种状态。
      • 直译：今后要处在“受您关照”的状态。→ 今后要多多麻烦您了。
    • どうぞ よろしく お願いします。
      • どうぞ：请（加强请求的语气）。
      • よろしく：请“好好对待我/多关照我”，是副词形。
      • お願いします：拜托了、拜托您。
      • 整体慣用意义：以后请多多关照。（自我介绍、加入新环境时的万能句。）

  • 对对方“よろしく”的回应

    • こちらこそ よろしく。
      • こちら：这边、我这边（比“わたし”更客气、更抽象）。
      • こそ：强调助词，“正是……才……”。
      • 字面：正是我这边才要请你多多关照。
      • 含义：哪里哪里，我才是要请您关照 —— 用来回应别人说「よろしく お願いします」。

  • 送礼物时的说法

    • あのう、これ、ほんの 気持ちです。
      • あのう：那个……（犹豫+礼貌引出话题）。
      • これ：这个（手上的礼物）。
      • ほんの：仅仅、只不过（带有“很小、没什么”的语气）。
      • 気持ち：心意、心情。
      • 字面：这个，只是“一点点心意”。
      • 语用：
        • 刻意“贬低”礼物，以示谦虚，避免给对方压力。
        • 非常日式的礼貌：强调“东西不贵重，只是表达心意”。

  • 收礼物时的自然反应

    • あ、どうも……。何ですか。
      • あ：啊（反应声）。
      • どうも：这里近似“哎呀，谢谢”，略带惊喜/不好意思。
      • 何ですか：这是什么呢？
      • 合起来：啊，谢谢……这是什么呀？—— 一边接受，一边好奇询问内容。

  • 说明礼物 & 正式致谢

    • コーヒーです。どうぞ。
      • コーヒーです：是咖啡（名词句）。
      • どうぞ：请收下 / 请用。
    • どうも ありがとう ございます。
      • ありがとう：谢谢。
      • ございます：敬语形式，使表达更郑重。
      • どうも：非常、真是。
      • 合起来：真是非常感谢您。

3. 相关文化背景小贴士

  • 「お世話になります」文化

    • 日语里的「お世話になる」字面是“受到关照/照顾”。
    • 在日本，进入一个新的“圈子”（公司、学校、公寓、小组）时，会预设“以后会麻烦大家”，提前表达感谢和歉意。
    • 所以「これから お世話になります」同时包含：
      • “今后可能会给您添麻烦”的歉意；
      • “请多多指教/帮助”的请求；
      • “很期待和您建立良好关系”的友好姿态。

  • 「よろしく お願いします」的广泛用法

    • 不只“初次见面”用，在很多场景都会出现：
      • 邮件结尾：今後とも よろしく お願いします。
      • 拜托别人做事：明日までに お願い できますか。よろしく お願いします。
    • 中文很难用一个固定句对应，大致涵盖：请多关照 / 拜托了 / 以后也请多多指教 等多重含义。

  • 送礼时“降格”的习惯

    • 日本人送礼时，常常会故意用“降低礼物价值”的说法：
      • つまらない もの ですが……（虽然是个没什么价值的东西……）。
      • ほんの 気持ちです（只是一点点心意）。
    • 这不是看不起礼物，而是通过自谦：
      • 避免让对方有“收了这么贵重的东西不好意思”的心理负担；
      • 表现出“重视关系 > 礼物本身”的价值观。

  • 邻里关系 & 初次见面

    • 在公寓/宿舍里，新搬来的人主动上门打招呼、送点小礼物，是日式“建立邻里关系”的理想范式：
      • 自报姓名和房间号（408の サントスです）；
      • 说明“今后要多关照”（これから お世話になります）；
      • 递上小礼物（これ、ほんの 気持ちです）。
    • 对方则用「こちらこそ よろしく」「どうも ありがとう ございます」来接住这种好意。

2. 句子公式总结

  • 这里是～：

    • ここは N です。→ 这里是 N。

  • ～在那边：

    • N は あそこです。→ N 在那边。

  • 询问“哪里”：

3. 语法与用法要点

  • ここ／そこ／あそこ vs どこ

    • ここ：这里（靠近说话人）。
    • そこ：那里（靠近听话人）。
    • あそこ：那里（离双方都远的地方）。
    • どこ：哪里（疑问）。
    • 结构：場所 は ここ／そこ／あそこ／どこ です。

  • どこ → どちら 的礼貌升级

    • どちら：比 どこ 更礼貌的“哪边”。
    • 用在：
      • 询问方向：エレベーターは どちらですか。（= どこですが，更客气。）
      • 询问国家/出身：[お]国は どちらですか。（您来自哪个国家？）

  • [お]国 的「お」

    • 「お」：尊敬/美化前缀，加在和对方有关的名词前面更礼貌。
    • 国 → お国：对他人国家的尊称，类似“贵国”。

  • どこの N：来源、产地

    • どこの N：哪里的 N / 哪个地方的 N。
    • 可以用来问：商品品牌、产地、学校、公司等。
    • 本课例句：それは どこの 靴ですか。→ 那是哪里产的鞋？

  • N1 の N2：所属 / 产地 / 范围

    • 新大阪の 駅（新大阪车站）、イタリアの 靴（意大利的鞋）、日本の 会社（日本的公司）。
    • 根据名词关系不同，可以翻译成“……的……”、 “来自……”等。

  • いくら：价格专用疑问词

    • いくらですか：多少钱？
    • 前面可以加指示词或名词短语：
      • この 時計は いくらですか。（这块表多少钱？）
      • その 靴は いくらですか。（那双鞋多少钱？）

  • です／そうです 的回答

    • 是/不是某地：
      • ここは 新大阪ですか。→ はい、そうです。（是的，是新大阪。）
    • 「そうです」：肯定对方刚刚说的内容，相当于“没错”。

2. 句子功能 & 结构拆解

  • 问卖场在哪里

    • すみません。ワイン売り場は どこですか。
      • すみません：不好意思，打扰一下（起手式）。
      • ワイン売り場：葡萄酒卖场/柜台。
      • 〜は どこですか：～在哪里？

  • 店员礼貌回答

    • 地下１階で ございます。
      • 地下１階で：在地下一层。
      • ございます：「あります」的敬语形，服务行业常用，比「あります」更客气。

  • 简短致谢

    • どうも。
      • 这里相当于“谢谢”，语气比「ありがとうございます」轻一些但仍礼貌。

  • 指定要看的商品

    • すみません。その ワインを 見せて ください。
      • その ワイン：那瓶酒（靠近店员的那一瓶）。
      • Nを 見せて ください：请给我看 N；购物时常用来请店员拿下来看看。

  • 店员递给客人

    • はい、どうぞ。
      • 「给你 / 请用」的固定说法，递东西、请对方先用时都会用。

  • 确认产地

    • これは フランスの ワインですか。
      • これは：这瓶。
      • フランスの ワイン：法国的酒（法国葡萄酒）。
      • 〜ですか：是……吗？

  • 否定+更正

    • いいえ、イタリアのです。
      • いいえ：不。
      • イタリアの（ワイン）です：是意大利的（酒）。名词“ワイン”省略，用「の」代替。

  • 问价格 & 做决定

    • いくらですか。→ 2,500円です。
      • いくらですか：多少钱？
      • 2,500円です：2500日元。
    • じゃ、これを ください。
      • じゃ：那么、那就……
      • これを ください：请给我这个（就买这瓶）。

3. 小小文化背景

  • すみません 的多重含义

    • 不只是“对不起”，还兼有“打扰一下”“不好意思，麻烦你一下”的意思。
    • 问路、叫店员、递盘子时都可以先说「すみません」。

  • 服务行业里的敬语

    • 店员对顾客通常用「ございます」而不是「あります」，显得更专业、有礼貌：
      • 地下１階で ございます。
    • 递东西时用「はい、どうぞ。」而不是简单的「はい。」

  • 买东西时的“下单用语”

    • 看完、问完价格以后，不说“买这个”，而常说：
      • じゃ、これを ください。（那就请给我这个。）
    • 既表达决定购买，又保持礼貌和柔和的语气，是日本购物场景中的常用句。

    2) 分（分 - ふん／ぷん）

    数字 日语（汉字） 读音（平假名） 规律

    • 规律总结:

      • pun (ぷん): 1, 3, 4, 6, 8, 10 分结尾（通常伴随促音或拨音）。

      • 30分也可以说 半（はん）。

    3) 24小时制与 AM/PM

    • 午前（ごぜん） / 午後（ごご）: 放在时间前面。
      • 例: 午前 7時 (上午7点)，午後 3時 (下午3点)。
    • 24小时制: 常用于时刻表、商务场合。
      • 下午1点 = 13時 (じゅうさんじ)。
      • 下午10点 = 22時 (にじゅうにじ)。
      • 午夜 = 0時 (れいじ)。

  • 动词的活用（Masu形）

    • 日语动词用来做谓语时，礼貌体以 ます 结尾。
    • ～ます (肯定，非过去): 描述现在的习惯动作或将来的动作。例如：毎日勉強します (每天学习)，明日勉強します (明天学习)。
    • ～ません (否定，非过去): 勉強しません。
    • ～ました (肯定，过去): きのう勉強しました (昨天学习了)。
    • ～ませんでした (否定，过去): きのう勉強しませんでした。

  • 助词「に」：动作的时间点

    • 表示动作发生的具体时间点（含有数字的时间）。
    • 例：6時に 起きます (6点起床)。
    • 注意：今、きのう、きょう、あした、毎朝 等词后面不加 に。
    • 星期几后面可加可不加 に：日曜日(に) 勉強します。

  • 助词「から」和「まで」：起点和终点

    • から (From): 时间或空间的起点。
    • まで (Until/To): 时间或空间的终点。
    • 既可以连用 9時から 5時まで，也可以单独使用 9時から 働きます。
    • 也可以直接做谓语：銀行は 9時から 3時までです (银行是从9点到3点)。

      • 回答模板：X時までです。

  • 追问休息日

    • 休みは 何曜日ですか。
      • 休み：这里指“固定休息日”。
      • 何曜日：问具体星期。

  • 回答休息日

    • 日曜日です。
      • 本段会话中的具体答案是“星期日休息”。

  • 收尾

    • そうですか。どうも。
      • そうですか：表示“了解了”。
      • どうも：简短致谢，口语里很常见。

3. 场景化使用提示

  • 这段会话展示了“先问号码再致电店铺”的常见流程。
  • 新版里最核心两问是：
    • そちらは 何時までですか。
    • 休みは 何曜日ですか。
  • 店方常见回答格式：
    • X時までです。
    • ～曜日です。

4. 小小文化背景

  • 电话开场先说「すみません」

    • 在电话里常用来表达“打扰了”，语气更礼貌。
    • 不只是“对不起”，也有“劳驾”的意思。

  • 「そちら」的礼貌距离感

    • 在电话里直接说「あなた」通常不自然，容易显得生硬。
    • 用「そちらは 何時までですか」更礼貌、更自然。

  • 电话号码分段读法

    • 教材里常用「の」分隔号码（5275の 2725），便于听写和复述。
    • 实际通话中，听不清时可以复述确认一次。

  • 通话结尾致谢

    • 问完营业时间和休息日后，用「そうですか。どうも。」收尾很自然。
    • 如果场景更正式，也可说「どうも ありがとうございます」。

第五课

textbook: https://ttrw.jp/static/textbook//grammer_free/22/dr1-5.pdf

文型 (ぶんけい - bunkei)

  1. わたしは　京都へ　行きます。
  2. わたしは　タクシーで　うちへ　帰ります。
  3. わたしは　家族と　日本へ　来ました。

文型小结

0. 本课核心（三句）

功能 句型 例句
表示移动的目的地 场所 + へ + 行きます／来ます／帰ります わたしは 京都へ 行きます。
表示交通手段 交通工具 + で + 行きます／来ます／帰ります わたしは タクシーで うちへ 帰ります。
表示同行者 人 + と + 行きます／来ます／帰ります わたしは 家族と 日本へ 来ました。

1. 单词表

日语 读音 中文意思
行きます いきます 去
来ます きます 来
帰ります かえります 回去、回来
京都 きょうと 京都
うち うち 家
家族 かぞく 家人、家属
日本 にほん 日本
タクシー たくしー 出租车
へ へ（读作 え） 表示移动方向、目的地
で で 表示方式、手段
と と 和、与

2. 语法点剖析

  • 移动动词：行きます／来ます／帰ります

    • 这三个动词是本课中心。
    • 行きます：从说话人所在的位置去别处。
    • 来ます：朝说话人所在的位置来，或说话人视角中的“来到某地”。
    • 帰ります：回到原来的地方、自己的家、宿舍等。

  • 助词「へ」：移动的方向 / 目的地

    • 结构：场所 + へ + 行きます／来ます／帰ります
    • 例：京都へ 行きます、日本へ 来ました、うちへ 帰ります
    • 这里的「へ」写作 へ，实际读作 え。
    • 它强调“朝某个方向移动”，是本课最重要的助词之一。

  • 助词「で」：交通工具 / 手段

    • 结构：交通工具 + で + 行きます／来ます／帰ります
    • 例：タクシーで 帰ります、新幹線で 行きます
    • 表示“乘什么去”。
    • 注意：走路时一般说 歩いて 行きます，不用 足で 行きます。

  • 助词「と」：同行者

    • 结构：人 + と + 行きます／来ます／帰ります
    • 例：家族と 日本へ 来ました、山田さんと 東京へ 行きます
    • 表示“和谁一起去”。
    • 如果想表达“一个人去”，通常说 ひとりで 行きます，不用 だれも と。

  • 疑问词「どこ」「何で」「だれと」「いつ」

    • どこへ 行きますか：去哪里？
    • 何で 行きますか：坐什么去？ / 用什么方式去？
    • だれと 行きますか：和谁一起去？
    • いつ 来ましたか：什么时候来的？

  • 日期 + に

    • 具体日期后面一般加 に。
    • 例：3月25日に 来ました。
    • 但问句里 いつ 后面不加 に：いつ 日本へ 来ましたか。
    • 生日也常直接说：6月13日です。

3. 常见错误对照表

常见错误 正确说法 说明
京都に行きます 京都へ行きます 本课重点先掌握教材里的 へ 表示移动方向
タクシーを帰ります タクシーで帰ります 交通工具用 で
家族で日本へ来ました 家族と日本へ来ました 同行者用 と
いつに日本へ来ましたか いつ 日本へ 来ましたか いつ 后面一般不加 に
3月25日 来ました 3月25日に来ました 具体日期后一般加 に

例文 (れいぶん - reibun)

  1. あした　どこへ　行きますか。
    ･･･奈良へ　行きます。

  2. 日曜日　どこ［へ］も　行きましたか。
    ･･･どこ［へ］も　行きませんでした。

  3. 何で　東京へ　行きますか。
    ･･･新幹線で　行きます。

  4. だれと　東京へ　行きますか。
    ･･･山田さんと　行きます。

  5. いつ　日本へ　来ましたか。
    ･･･3月25日に　来ました。

  6. 誕生日は　いつですか。
    ･･･6月13日です。

例文小结

0. 例文练习目标

目标 代表句型
询问去哪里 どこへ 行きますか。
询问是否去了某处 どこへも 行きませんでした。
询问交通方式 何で 行きますか。
询问同行者 だれと 行きますか。
询问到达时间 いつ 来ましたか。
询问日期 / 生日 誕生日は いつですか。

1. 生词对照表

日语 读音 中文意思
あした あした 明天
奈良 なら 奈良
日曜日 にちようび 星期日
何で なんで 用什么；坐什么
東京 とうきょう 东京
新幹線 しんかんせん 新干线
だれと だれと 和谁
山田さん やまださん 山田先生 / 女士
いつ いつ 什么时候
誕生日 たんじょうび 生日
3月25日 さんがつ にじゅうごにち 3月25日
6月13日 ろくがつ じゅうさんにち 6月13日

2. 句子公式总结

功能 提问模板 回答模板 例句
询问目的地 どこへ 行きますか。 Nへ 行きます。 あした どこへ 行きますか。→ 奈良へ 行きます。
询问是否去了哪里 どこへも 行きましたか。 どこへも 行きませんでした。 日曜日 どこへも 行きましたか。→ どこへも 行きませんでした。
询问交通方式 何で Nへ 行きますか。 交通工具で 行きます。 何で 東京へ 行きますか。→ 新幹線で 行きます。
询问同行者 だれと Nへ 行きますか。 Nと 行きます。 だれと 東京へ 行きますか。→ 山田さんと 行きます。
询问来某地的时间 いつ Nへ 来ましたか。 日期に 来ました。 いつ 日本へ 来ましたか。→ 3月25日に 来ました。
询问日期 Nは いつですか。 日期です。 誕生日は いつですか。→ 6月13日です。

3. 语法补充（例文里要特别注意）

  • 「どこへも 行きませんでした」

    • どこも / どこへも 和否定形式一起用，表示“哪里也没去”。
    • 是初学者很常见的固定搭配。

  • 「何で」在本课不是“为什么”

    • 这里的 何で 读作 なんで，意思是“用什么 / 坐什么”。
    • 因为后面接的是移动动词，所以表示交通手段。

  • 日期和生日的说法

    • 3月25日に 来ました：到来的具体日期。
    • 誕生日は 6月13日です：生日这种“日期信息”常直接用 です 说明。

会话 (かいわ - kaiwa)

甲子園へ　行きますか

サントス：すみません。甲子園まで　いくらですか。
女の人：350円です。
サントス：350円ですね。ありがとう　ございました。
女の人：どう　いたしまして。

サントス：すみません。甲子園は　何番線ですか。
駅員：5番線です。
サントス：どうも。

サントス：あのう、この　電車は　甲子園へ　行きますか。
男の人：いいえ。次の　「普通」ですよ。
サントス：そうですか。どうも。

会话小结

这段会话是“问票价 → 问站台 → 确认列车是否到达目的地”的连续乘车场景，重点是交通出行中的提问表达。

1. 生词 & 固定表达表格

日语表达 读音 类型 含义/用法说明
甲子園 こうしえん 专有名词 甲子园；地名 / 车站名
～まで ～まで 助词 到～为止；到某地
いくらですか いくらですか 句型 多少钱
350円ですね さんびゃくごじゅうえんですね 句型 350日元，对吧 / 我知道了是350日元
ありがとう ございました ありがとう ございました 固定表达 非常感谢（过去式，更礼貌）
どう いたしまして どう いたしまして 固定表达 不客气
何番線ですか なんばんせんですか 句型 是几号站台 / 几号线
駅員 えきいん 名词 站务员
電車 でんしゃ 名词 电车
普通 ふつう 名词 普通列车
次の つぎの 连体词 下一个 / 下一班
～ですよ ～ですよ 句尾 告知、提醒对方的语气

2. 句子功能 & 结构拆解

  • 询问到某地多少钱

    • すみません。甲子園まで いくらですか。
      • 甲子園まで：到甲子园。
      • いくらですか：多少钱。
      • 合起来就是：到甲子园多少钱？

  • 听到价格后的确认

    • 350円ですね。
      • ですね 在这里有“我确认一下”“原来是这样”的语气。

  • 更礼貌的致谢

    • ありがとう ございました。
      • 比 ありがとう 更礼貌，服务场景里很常见。

  • 询问站台

    • すみません。甲子園は 何番線ですか。
      • 何番線：几号站台 / 几号线。
      • 是乘车场景非常常用的问法。

  • 确认这趟车是否到目的地

    • あのう、この 電車は 甲子園へ 行きますか。
      • この 電車：这趟电车。
      • 甲子園へ 行きますか：去甲子园吗？

  • 对方否定并补充正确信息

    • いいえ。次の「普通」ですよ。
      • 不是这班。
      • 次の：下一班。
      • 普通：普通列车。
      • ですよ：提醒、告知对方“请注意这个信息”。

  • 听懂后的自然收尾

    • そうですか。どうも。
      • 表示“原来如此，谢谢”。

3. 场景化使用提示

  • 这段会话可以直接用于：

    • 问车费
    • 问站台
    • 问某辆车去不去目的地

  • 本课出行场景的三个核心问句：

    • Nまで いくらですか。
    • Nは 何番線ですか。
    • この 電車は Nへ 行きますか。

  • 常见回答格式：

    • 350円です。
    • 5番線です。
    • いいえ。次の 普通です。

  • 小小文化背景

  • 日本车站里常见的列车种类

    • 普通：每站都停的普通列车。
    • 以后还会见到 急行、特急、快速 等。
    • 所以搭车前确认“这是不是我要坐的车”非常重要。

  • 问路时的礼貌起手式

    • すみません 和 あのう 都很常用。
    • すみません 更直接，用来叫住对方。
    • あのう 更柔和，适合在正式或陌生场合开口前缓冲一下。

  • ありがとう ございました 的感觉

    • 这是对已经发生的帮助表示感谢。
    • 比 ありがとう 更礼貌，也比 どうも 更完整。
