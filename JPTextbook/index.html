第二课

textbook: https://ttrw.jp/static/textbook//grammer_free/19/dr1-2.pdf

文型 (ぶんけい - bunkei)

これは　辞書です

これは　コンピューターの　本です

それは　私の　傘です

この　傘は　私のです

  • これは／それは／あれは

    • これ = 这（在说话人身边）
    • それ = 那（在听话人身边）
    • あれ = 那（离双方都远）

  • の（名词的修饰关系）

    • 「の」 (Content)
      • これ は コンピューターの 本 です。
      • This is a book on computers (a computer book).
      • It tells us what kind of object Noun 2 is, or what its contents are.
    • 「の」 (Possession)

      • それ は 私の 傘 です。

      • That is my umbrella.

      • This is the most common use of the particle 「の」. It indicates ownership. Noun 1 owns Noun 2.

例文 (れいぶん - reibun)

これはテレホンカードですか。

･･･はい、そうです。


それは　ノートですか。

･･･いいえ、そうじゃありません。手帳です。


それは 何ですか。

･･･[これは] 名刺です。


これは「9」ですか、「7」ですか。

･･･「9」です。


それは何の雑誌ですか。

･･･自動車の雑誌です。


あれは　だれの　かばんですか。

･･･佐藤さんのかばんです。


この傘はあなたのですか。

･･･いいえ、わたしのじゃありません。


このかぎはだれのですか。

･･･わたしのです。

例文小结

1. 生词对照表（例文中出现的新词）

日语 读音（假名） 中文意思

ノート のーと 笔记本
手帳 てちょう 记事本、手帐
名刺 めいし 名片
雑誌 ざっし 杂志
自動車 じどうしゃ 汽车
かばん かばん 包、书包
傘 かさ 伞
かぎ かぎ 钥匙
だれ だれ 谁
何 なに／なん 什么

2. 句子公式总结（按用途记）

  • 判断 + 提问：
    • これは N ですか。→ 这是 N 吗？
    • それは N ですか。→ 那是 N 吗？
  • 选择疑问句：
    • A ですか、B ですか。→ 是 A 还是 B？
    • これは「9」ですか、「7」ですか。
  • 询问事物是什么：
    • それは 何ですか。→ 那是什么？
    • それは 何の N ですか。→ 那是「什么的」N / 关于什么的 N？
  • 询问所属/归属：
    • あれは だれの N ですか。→ 那是谁的 N？
    • この N は あなたのですか。→ 这 N 是你的吗？
    • この N は だれのですか。→ 这 N 是谁的？
  • 肯定/否定回答模板：
    • はい、そうです。→ 是的，是这样。
    • いいえ、そうじゃありません。N です。→ 不，不是，是 N。
    • いいえ、わたしのじゃありません。→ 不，不是我的。
    • わたしのです。→ 是我的。

3. 语法点剖析

  • 指示词 + は：これは／それは／あれは

    • これ：这（在说话人身边）。
    • それ：那（在听话人身边）。
    • あれ：那（离双方都远）。
    • 后面接「は」做主题：これはテレホンカードですか。

  • 助词「の」的两大用法

    • 所有/所属：
      • だれの かばん（谁的包）、わたしの かばん（我的包）、佐藤さんの かばん（佐藤先生的包）。
      • 结构：N1 の N2 → “N1 的 N2”。
    • 内容/类别：
      • 自動車の 雑誌：汽车杂志（内容是“汽车”）。
      • コンピューターの 本：计算机的书（内容是“计算机”）。

  • 疑问词「何」「だれ」+ の

    • 何の N：什么的 N / 关于什么的 N
      • それは 何の 雑誌ですか。→ 那是关于什么的杂志？
    • だれの N：谁的 N
      • あれは だれの かばんですか。→ 那是谁的包？

  • 「の」的代词化（代替名词）

    • この傘は あなたのです。
      • 表面意思：这把伞是你的。
      • 实际结构：あなたの（傘）です，「の」代替前面的「傘」，避免重复名词。
    • このかぎは だれのですか。→ わたしのです。
      • 问句里的「だれの（かぎ）」和回答里的「わたしの（かぎ）」都省略了「かぎ」，靠「の」来代替。

  • 否定表达「〜じゃありません」

    • 相当于更正式的「〜ではありません」，是否定「〜です」的礼貌说法。
    • 例：いいえ、そうじゃありません。手帳です。→ 不，不是（笔记本），是记事本。

会话小结（表达梳理）

1. 生词 & 固定表达表格

コーヒーです。どうぞ。 コーヒーです。どうぞ。 句型 是咖啡，请收下 / 请用

2. 句子功能 & 结构拆解

  • 开场确认对方身份

    • はい。どなたですか。
      • はい：喂 / 是（接电话、应答时的开场）。
      • どなた：礼貌版“谁”。
      • 〜ですか：礼貌疑问结尾。
      • 合起来：喂，请问是哪位？—— 标准“接电话/开门问身份”的表达。

  • 报上房间号 + 姓名

    • 408の サントスです。
      • N1 の N2：N1 的 N2（所属关系）。
      • 408の サントス：408 房间的 Santos（住在 408 的那位 Santos）。
      • 〜です：礼貌判断句。
      • 场景：在公寓/宿舍里表明“我是 X 房间的某某”。

  • 正式见面 + 建立关系

    • こんにちは。サントスです。
      • こんにちは：白天打招呼“你好”。
      • サントスです：我是 Santos（名词＋です）。
    • これから お世話に なります。
      • これから：今后、从现在起。
      • お世話：关照、照顾。
      • 〜に なります：变成某种状态。
      • 直译：今后要处在“受您关照”的状态。→ 今后要多多麻烦您了。
    • どうぞ よろしく お願いします。
      • どうぞ：请（加强请求的语气）。
      • よろしく：请“好好对待我/多关照我”，是副词形。
      • お願いします：拜托了、拜托您。
      • 整体慣用意义：以后请多多关照。（自我介绍、加入新环境时的万能句。）

  • 对对方“よろしく”的回应

    • こちらこそ よろしく。
      • こちら：这边、我这边（比“わたし”更客气、更抽象）。
      • こそ：强调助词，“正是……才……”。
      • 字面：正是我这边才要请你多多关照。
      • 含义：哪里哪里，我才是要请您关照 —— 用来回应别人说「よろしく お願いします」。

  • 送礼物时的说法

    • あのう、これ、ほんの 気持ちです。
      • あのう：那个……（犹豫+礼貌引出话题）。
      • これ：这个（手上的礼物）。
      • ほんの：仅仅、只不过（带有“很小、没什么”的语气）。
      • 気持ち：心意、心情。
      • 字面：这个，只是“一点点心意”。
      • 语用：
        • 刻意“贬低”礼物，以示谦虚，避免给对方压力。
        • 非常日式的礼貌：强调“东西不贵重，只是表达心意”。

  • 收礼物时的自然反应

    • あ、どうも……。何ですか。
      • あ：啊（反应声）。
      • どうも：这里近似“哎呀，谢谢”，略带惊喜/不好意思。
      • 何ですか：这是什么呢？
      • 合起来：啊，谢谢……这是什么呀？—— 一边接受，一边好奇询问内容。

  • 说明礼物 & 正式致谢

    • コーヒーです。どうぞ。
      • コーヒーです：是咖啡（名词句）。
      • どうぞ：请收下 / 请用。
    • どうも ありがとう ございます。
      • ありがとう：谢谢。
      • ございます：敬语形式，使表达更郑重。
      • どうも：非常、真是。
      • 合起来：真是非常感谢您。

3. 相关文化背景小贴士

  • 「お世話になります」文化

    • 日语里的「お世話になる」字面是“受到关照/照顾”。
    • 在日本，进入一个新的“圈子”（公司、学校、公寓、小组）时，会预设“以后会麻烦大家”，提前表达感谢和歉意。
    • 所以「これから お世話になります」同时包含：
      • “今后可能会给您添麻烦”的歉意；
      • “请多多指教/帮助”的请求；
      • “很期待和您建立良好关系”的友好姿态。

  • 「よろしく お願いします」的广泛用法

    • 不只“初次见面”用，在很多场景都会出现：
      • 邮件结尾：今後とも よろしく お願いします。
      • 拜托别人做事：明日までに お願い できますか。よろしく お願いします。
    • 中文很难用一个固定句对应，大致涵盖：请多关照 / 拜托了 / 以后也请多多指教 等多重含义。

  • 送礼时“降格”的习惯

    • 日本人送礼时，常常会故意用“降低礼物价值”的说法：
      • つまらない もの ですが……（虽然是个没什么价值的东西……）。
      • ほんの 気持ちです（只是一点点心意）。
    • 这不是看不起礼物，而是通过自谦：
      • 避免让对方有“收了这么贵重的东西不好意思”的心理负担；
      • 表现出“重视关系 > 礼物本身”的价值观。

  • 邻里关系 & 初次见面

    • 在公寓/宿舍里，新搬来的人主动上门打招呼、送点小礼物，是日式“建立邻里关系”的理想范式：
      • 自报姓名和房间号（408の サントスです）；
      • 说明“今后要多关照”（これから お世話になります）；
      • 递上小礼物（これ、ほんの 気持ちです）。
    • 对方则用「こちらこそ よろしく」「どうも ありがとう ございます」来接住这种好意。

第三课

文型 (ぶんけい - bunkei)

ここは　食堂です。

電話は　あそこです。

例文 (れいぶん - reibun)

ここは　新大阪ですか。

･･･はい、そうです。


お手洗いは　どこですか。

･･･あそこです。


山田さんは　どこですか。

･･･事務所です。


エレベーターは　どちらですか。

･･･そちらです。


[お]国は　どちらですか。

･･･アメリカです。


それは　どこの　靴ですか。

･･･イタリアの　靴です。


この　時計は　いくらですか。

･･･18,600円です。

例文小结（第三课）

1. 生词对照表（地点・物品・价格）

日语 读音 中文意思
食堂 しょくどう 食堂
電話 でんわ 电话
あそこ あそこ 那里（离双方都远的地方）
お手洗い おてあらい 洗手间、厕所（礼貌说法）
事務所 じむしょ 事务所、办公室
エレベーター えれべーたー 电梯
どちら どちら 哪边；哪一个（どこ／どれ的礼貌版）
[お]国 （お）くに 贵国、您是哪国人（加「お」显得更尊敬）
靴 くつ 鞋
時計 とけい 表、钟
いくら いくら 多少钱（价格）
円 えん 日元

2. 句子公式总结

  • 这里是～：

    • ここは N です。→ 这里是 N。
    • 例：ここは 食堂です。（这里是食堂。）

  • ～在那边：

    • N は あそこです。→ N 在那边。
    • 例：電話は あそこです。（电话在那边。）

  • 询问“哪里”：

    • N は どこですか。→ N 在哪里？
    • 例：お手洗いは どこですか。（洗手间在哪里？）

  • 询问“哪边/哪一位”（礼貌）：

    • N は どちらですか。→ N 在哪边？/是哪一位？
    • 例：エレベーターは どちらですか。（电梯在哪边？）
    • [お]国は どちらですか。（您是哪里人？/您来自哪国？）

  • 询问“哪里产的 / 哪个地方的”：

    • それは どこの N ですか。→ 那是哪里产的 N？
    • 例：それは どこの 靴ですか。（那是哪里产的鞋？）

  • 说明出处 / 产地：

    • N1 の N2：N1 的 N2。
    • 例：イタリアの 靴です。（意大利的鞋。）

  • 询问价格：

    • この N は いくらですか。→ 这个 N 多少钱？
    • 例：この 時計は いくらですか。（这块表多少钱？）
    • 回答：金額＋円です。→ 18,600円です。（18,600 日元。）

3. 语法与用法要点

  • ここ／そこ／あそこ vs どこ

    • ここ：这里（靠近说话人）。
    • そこ：那里（靠近听话人）。
    • あそこ：那里（离双方都远的地方）。
    • どこ：哪里（疑问）。
    • 结构：場所 は ここ／そこ／あそこ／どこ です。

  • どこ → どちら 的礼貌升级

    • どちら：比 どこ 更礼貌的“哪边”。
    • 用在：
      • 询问方向：エレベーターは どちらですか。（= どこですが，更客气。）
      • 询问国家/出身：[お]国は どちらですか。（您来自哪个国家？）

  • [お]国 的「お」

    • 「お」：尊敬/美化前缀，加在和对方有关的名词前面更礼貌。
    • 国 → お国：对他人国家的尊称，类似“贵国”。

  • どこの N：来源、产地

    • どこの N：哪里的 N / 哪个地方的 N。
    • 可以用来问：商品品牌、产地、学校、公司等。
    • 本课例句：それは どこの 靴ですか。→ 那是哪里产的鞋？

  • N1 の N2：所属 / 产地 / 范围

    • 新大阪の 駅（新大阪车站）、イタリアの 靴（意大利的鞋）、日本の 会社（日本的公司）。
    • 根据名词关系不同，可以翻译成“……的……”、 “来自……”等。

  • いくら：价格专用疑问词

    • いくらですか：多少钱？
    • 前面可以加指示词或名词短语：
      • この 時計は いくらですか。（这块表多少钱？）
      • その 靴は いくらですか。（那双鞋多少钱？）

  • です／そうです 的回答

    • 是/不是某地：
      • ここは 新大阪ですか。→ はい、そうです。（是的，是新大阪。）
    • 「そうです」：肯定对方刚刚说的内容，相当于“没错”。

    练习（问路＋问价格）

    练习 1：问设施在哪儿

    场景：你在学校问别人图书馆、事务所在哪里。

    • A：すみません。図書館は　どこですか。

    • B：図書館は　あそこです。

    • A：事務所は　どこですか。

    • B：事務所は　ここです。

    练习 2：问电梯和洗手间

    • A：すみません。エレベーターは　どちらですか。

    • B：エレベーターは　そちらです。

    • A：お手洗いは　どこですか。

    • B：お手洗いは　あそこです。

    练习 3：问产地＋价格

    在百货公司看包和手表。

    • A：それは　どこの　かばんですか。

    • B：フランスの　かばんです。

    • A：この　かばんは　いくらですか。

    • B：15,000円です。

    • A：この　時計は　いくらですか。

    • B：25,800円です。

    练习 4：换成你自己的版本（中→日）

    试着自己翻译并写在下面：

    1. 请问，电梯在哪里？——在那边。
    2. 这双鞋是哪里产的？——是意大利的鞋。
    3. 这块表多少钱？——28,000日元。

会话 (かいわ - kaiwa)

これを　ください

マリア：すみません。ワイン売り場は　どこですか。

店員A：地下１階で　ございます。

マリア：どうも。

マリア：すみません。その　ワインを　見せて　ください。

店員B：はい、どうぞ。

マリア：これは　フランスの　ワインですか。

店員B：いいえ、イタリアのです。

マリア：いくらですか。

店員B：2,500円です。

マリア：じゃ、これを　ください。

会话小结（第三课）

1. 生词 & 固定表达表格

日语表达 读音 类型 含义/用法说明
ワイン売り場 わいんうりば 名词 卖葡萄酒的柜台、卖酒区（売り場＝卖场、柜台）
地下１階 ちか いっかい 名词 地下一层
ございます ございます 敬语动词 「あります」的礼貌形，更郑重，多用于服务行业
どうも どうも 副词/感叹 在这里相当于“谢谢”；也可作“总觉得、很”等
その N その N 指示词+名词 那个～（靠近听话人的东西）
Nを 見せて ください 〜を みせて ください 句型 请给我看～；“见せる（给……看）”+てください（请……）
はい、どうぞ はい、どうぞ 固定表达 好的，请（给你）；店员递东西时常用
これは N ですか これは N ですか 句型 这是 N 吗？
いいえ、Nのです いいえ、〜のです 句型 不，是～的；「の」代替前面的名词（如ワイン）
いくらですか いくらですか 句型 多少钱？（问价格）
じゃ、〜 じゃ、〜 连词 那么、那（就）～；常用来承接对方回答后的决定
これを ください これを ください 固定表达 请给我这个；用于买东西时下单、选定商品

2. 句子功能 & 结构拆解

  • 问卖场在哪里

    • すみません。ワイン売り場は どこですか。
      • すみません：不好意思，打扰一下（起手式）。
      • ワイン売り場：葡萄酒卖场/柜台。
      • 〜は どこですか：～在哪里？

  • 店员礼貌回答

    • 地下１階で ございます。
      • 地下１階で：在地下一层。
      • ございます：「あります」的敬语形，服务行业常用，比「あります」更客气。

  • 简短致谢

    • どうも。
      • 这里相当于“谢谢”，语气比「ありがとうございます」轻一些但仍礼貌。

  • 指定要看的商品

    • すみません。その ワインを 見せて ください。
      • その ワイン：那瓶酒（靠近店员的那一瓶）。
      • Nを 見せて ください：请给我看 N；购物时常用来请店员拿下来看看。

  • 店员递给客人

    • はい、どうぞ。
      • 「给你 / 请用」的固定说法，递东西、请对方先用时都会用。

  • 确认产地

    • これは フランスの ワインですか。
      • これは：这瓶。
      • フランスの ワイン：法国的酒（法国葡萄酒）。
      • 〜ですか：是……吗？

  • 否定+更正

    • いいえ、イタリアのです。
      • いいえ：不。
      • イタリアの（ワイン）です：是意大利的（酒）。名词“ワイン”省略，用「の」代替。

  • 问价格 & 做决定

    • いくらですか。→ 2,500円です。
      • いくらですか：多少钱？
      • 2,500円です：2500日元。
    • じゃ、これを ください。
      • じゃ：那么、那就……
      • これを ください：请给我这个（就买这瓶）。

3. 小小文化背景

  • すみません 的多重含义

    • 不只是“对不起”，还兼有“打扰一下”“不好意思，麻烦你一下”的意思。
    • 问路、叫店员、递盘子时都可以先说「すみません」。

  • 服务行业里的敬语

    • 店员对顾客通常用「ございます」而不是「あります」，显得更专业、有礼貌：
      • 地下１階で ございます。
    • 递东西时用「はい、どうぞ。」而不是简单的「はい。」

  • 买东西时的“下单用语”

    • 看完、问完价格以后，不说“买这个”，而常说：
      • じゃ、これを ください。（那就请给我这个。）
    • 既表达决定购买，又保持礼貌和柔和的语气，是日本购物场景中的常用句。
